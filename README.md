# Face Recognition Attendance System

## Features
- Real-time face recognition
- Automatic attendance marking
- Subject-wise tracking
- False detection prevention (buffer + lock)

## Tech Stack
- Python
- OpenCV
- DeepFace

## How to Run
1. Activate virtual environment
2. Install requirements
3. Run:
   python app.py

## Controls
- Press 'q' to exit

## Output
- Attendance stored in /logs folder

## Notes
- Requires good lighting
- Dataset should have multiple images per student
